# Screenshot Capture Guide for MyFinPal

## Quick Start

This guide will help you capture all required screenshots for App Store submission.

## Required Screenshots

You need screenshots for **3 iPhone sizes** (iPad is optional but recommended):

1. **iPhone 6.7"** (1290x2796px) - iPhone 14 Pro Max, 15 Pro Max
2. **iPhone 6.5"** (1242x2688px) - iPhone 11 Pro Max, XS Max  
3. **iPhone 5.5"** (1242x2208px) - iPhone 8 Plus
4. **iPad Pro 12.9"** (2048x2732px) - Optional but recommended since you support tablets

**Minimum:** 3 screenshots per device size  
**Recommended:** 5 screenshots per device size  
**Maximum:** 10 screenshots per device size

## Method 1: iOS Simulator (Easiest)

### Setup
1. Open your app in iOS Simulator
2. Choose the device size you want:
   - **iPhone 15 Pro Max** (for 6.7" screenshots)
   - **iPhone 11 Pro Max** (for 6.5" screenshots)
   - **iPhone 8 Plus** (for 5.5" screenshots)
   - **iPad Pro (12.9-inch)** (for iPad screenshots)

### Capture Steps
1. Navigate to the screen you want to capture
2. **Method A - Menu:**
   - Go to: **Device > Screenshots > [Device Name]**
   - Screenshot saved to Desktop
   
3. **Method B - Keyboard:**
   - Press `Cmd + S` (saves to Desktop)
   
4. **Method C - Right-click:**
   - Right-click on simulator window
   - Select "Save Screen"

### Resize (if needed)
Simulator screenshots are usually the correct size, but verify:
- Open screenshot in Preview (Mac)
- Check dimensions
- If wrong size, use Tools > Adjust Size to exact dimensions

## Method 2: Physical Device

### Capture
1. Take screenshot on your iPhone/iPad:
   - **iPhone X and later:** Side button + Volume Up
   - **iPhone 8 and earlier:** Home button + Side button
   
2. Transfer to computer:
   - AirDrop
   - iCloud Photos
   - Connect via USB and import

### Resize
1. Open in Preview (Mac) or image editor
2. Tools > Adjust Size
3. Set to exact dimensions:
   - 1290 x 2796 (iPhone 6.7")
   - 1242 x 2688 (iPhone 6.5")
   - 1242 x 2208 (iPhone 5.5")
   - 2048 x 2732 (iPad Pro 12.9")

## Tips

1. **Start with largest size** (iPhone 6.7") - easier to resize down than up
2. **Use same data** across all screenshots for consistency
3. **Test on device** - view screenshots on actual iPhone to ensure they look good
4. **Keep it clean** - no device frames, no status bars (unless part of design)
5. **High quality** - ensure screenshots are sharp and clear
6. **Update anytime** - you can update screenshots without a new submission
